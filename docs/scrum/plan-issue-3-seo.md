## Checklist por ciudad

Para cada una de las 10 ciudades, ejecutar estos pasos:

### Paso 1 — Investigación de keywords (30 min por ciudad)

- [ ] Buscar en Google Keyword Planner / Ubersuggest las keywords transaccionales del destino
- [ ] Identificar la keyword madre (mayor volumen + transaccional)
- [ ] Listar 5-15 keywords secundarias para fichas de yates
- [ ] Verificar búsquedas relacionadas ("renta de yate en [ciudad]", "paseo en yate [ciudad]", etc.)
- [ ] Guardar todo en la tabla maestra (ver formato abajo)

### Paso 2 — Asignación de keywords (15 min por ciudad)

- [ ] Asignar keyword madre a la página de ciudad
- [ ] Asignar 1 keyword única a cada ficha de yate de esa ciudad
- [ ] Verificar que ningún yate tenga la misma keyword que la página madre
- [ ] Verificar que ningún yate repita keyword con otro yate
- [ ] Si hay catamarán + velero + yate → considerar páginas por tipo de embarcación

### Paso 3 — Optimización on-page de la página madre (45 min por ciudad)

- [ ] H1 incluye keyword madre
- [ ] Title SEO comienza con keyword madre
- [ ] Meta description comienza con keyword madre
- [ ] Mínimo 4 H2 con keyword o variante
- [ ] Keyword mencionada al menos 7 veces en el texto
- [ ] Texto único de 300-500 palabras mínimo
- [ ] CTA principal: "Cotizar"
- [ ] Enlace interno a cada ficha de yate de esa ciudad
- [ ] Al menos 1 enlace externo útil

### Paso 4 — Optimización de fichas de yate (20 min por ficha)

- [ ] Title SEO con keyword única del yate
- [ ] H1 con keyword única
- [ ] Meta description con keyword
- [ ] Descripción completa y única (no copy-paste)
- [ ] Enlace interno a la página de ciudad
- [ ] Fotos optimizadas (alt text con keyword)
- [ ] Todos los campos llenos (pasajeros, precio, duración, tipo)

### Paso 5 — Verificación (10 min por ciudad)

- [ ] Buscar en Google (ventana incógnita) la keyword madre
- [ ] Verificar que la página correcta aparece (no canibalización)
- [ ] Revisar Google Search Console después de 1-2 semanas
