# Yatezzitos Global

**Tecnología, confianza y operaciones para reservar experiencias náuticas privadas de lujo.**

> **Confidencial · Uso interno**
>
> Este repositorio y su documentación están destinados exclusivamente para uso interno del proyecto **Yatezzitos Global**. No subir credenciales, datos privados de clientes, respaldos completos, tokens, llaves API ni información sensible del negocio.

---

## Visión general

**Yatezzitos Global** es el proyecto tecnológico y operativo de **Yatezzitos**, una marca enfocada en la renta de yates y experiencias náuticas privadas de lujo.

Hoy operamos en México y estamos construyendo las bases para escalar de **México al mundo**, combinando:

- turismo náutico privado,
- automatización operativa,
- CRM,
- posicionamiento SEO,
- inteligencia artificial,
- y una futura web app / app móvil para clientes, socios comerciales y equipo interno.

Yatezzitos no busca ser solo un sitio web de renta de yates.  
Busca convertirse en una **plataforma tecnológica de turismo náutico privado**, útil tanto para turistas como para propietarios, administradores, brokers, agencias y aliados del sector.

---

## Concepto de marca

**El arte del placer**.

Para Yatezzitos, esto significa diseñar experiencias náuticas privadas sin fricción, sin estrés innecesario y con la mayor confianza posible para el cliente final.

Nuestra filosofía es simple:

- lo que el cliente reserva debe ser lo que recibe,
- la experiencia debe sentirse clara, cómoda y confiable,
- y el proceso de renta debe ser mucho más sencillo que el modelo tradicional del mercado.

---

## Misión

Servir a las personas a través de experiencias turísticas náuticas privadas de lujo, con una atención útil, confiable, rápida, personalizada y cada vez más automatizada.

Yatezzitos existe para ayudar a que turistas, familias, parejas, grupos y viajeros premium puedan reservar embarcaciones privadas de forma sencilla, segura y eficiente, cuidando cada detalle de su experiencia.

---

## Visión

Convertirnos en la plataforma líder de renta de yates y experiencias náuticas privadas en el mundo.

A largo plazo, Yatezzitos busca evolucionar hacia un ecosistema más amplio que incluya:

- marketplace internacional de embarcaciones,
- motor de reservas conectado a disponibilidad real,
- herramientas tecnológicas para propietarios,
- automatización comercial y operativa,
- membresías vacacionales,
- y una futura infraestructura que impulse experiencias náuticas más limpias, eficientes y sostenibles.

La visión final es construir una especie de **cadena global de hospitalidad náutica privada**, con estándares de servicio, tecnología y confianza comparables a las mejores marcas del turismo premium mundial.

---

## Qué problema resolvemos

El mercado náutico privado tiene varios problemas estructurales:

- falta de confianza al reservar en internet,
- procesos poco claros o demasiado manuales,
- experiencias que no siempre coinciden con lo prometido,
- desorganización entre propietarios, administradores, brokers y clientes,
- seguimiento comercial deficiente,
- baja integración tecnológica entre disponibilidad, reservas, atención y operación.

Yatezzitos busca resolver esto con una mezcla de:

- atención humana de alto nivel,
- procesos claros,
- tecnología útil,
- automatización,
- y una futura capa de inteligencia artificial aplicada al turismo náutico.

---

## Diferenciador

A diferencia de un broker tradicional o de un simple marketplace, Yatezzitos busca diferenciarse por:

- atención privada y personalizada,
- acompañamiento antes, durante y después de la reserva,
- capacidad de resolver necesidades especiales del cliente,
- enfoque premium y orientado al detalle,
- integración entre marketing, ventas, reservas y seguimiento,
- visión tecnológica para escalar operación y confianza.

No solo buscamos rentar una embarcación.  
Buscamos ayudar a que la experiencia completa salga bien.

## Estado actual del proyecto

Actualmente, Yatezzitos se encuentra en una etapa de:

- negocio funcionando,
- web funcionando,
- automatizaciones parciales,
- MVP tecnológico construido sobre WordPress,
- operación activa con **8 años de experiencia** en el mercado.

### Stack actual

- **WordPress**
- **Elementor**
- **Tema Houzez** adaptado al negocio náutico
- **GoHighLevel** como centro operativo del CRM
- **Hostinger** como hosting actual
- Integraciones vía **webhooks**
- **Twilio** para verificación OTP en buscador
- Integraciones con **OpenAI** y herramientas de IA complementarias
- Plugin y endpoints personalizados para mostrar información dinámica de cotizaciones, reservas y páginas relacionadas

### Qué sí existe hoy

- Sitio web activo e indexado
- Operación nacional en México
- Buscador con verificación OTP
- Sistema actual de cotización
- Flujo de reserva con anticipo del 50%
- Visualización de información de cotización / reserva en páginas dinámicas
- CRM con automatizaciones parciales en GoHighLevel
- Base comercial operativa real

### Qué falta construir o mejorar

- rediseño completo del sitio según Figma,
- automatización total del embudo,
- mejor organización del CRM,
- mejor estructura de propietarios y clientes,
- calendario con disponibilidad en tiempo real,
- onboarding real de socios comerciales,
- área funcional para cuentas de clientes y propietarios,
- paneles más sólidos,
- mejor integración web + CRM + SEO + reservas,
- futura web app y app móvil.

- país,
- ciudad,
- tipo de embarcación,

Actualmente, **GoHighLevel** funciona como el centro operativo del CRM.
